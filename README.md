# Java-Interfaceler-Abstract-Classlar-ve-Inner-Classlar
Bu depo, Java programlama dilinde **nesne yönelimli programlama (OOP)** kavramlarını pekiştirmek amacıyla hazırlanmış örnek projeleri içermektedir.  
Projelerde özellikle aşağıdaki konular üzerinde durulmuştur:

- **Interface (Arayüzler)**
- **Abstract Class (Soyut Sınıflar)**
- **Inner Class (İç Sınıflar)**


---

## Örnek Projeler
- Yurtdışı çıkış programı
- Geometrik şekillerin alanlarını hesaplama
- Matematik ve Fizik Problemleri
